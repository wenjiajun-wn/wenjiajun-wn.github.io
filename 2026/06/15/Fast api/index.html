FastAPI 入门学习笔记

学习来源：DataCamp《Introduction to FastAPI》课程

什么是 FastAPI？

FastAPI 是一个现代、高性能的 Python Web 框架，专门用于构建 API。它由 Sebastián Ramírez 开发，基于两个强大的库：

  • Starlette：提供异步 Web 框架的核心能力
  • Pydantic：负责数据验证与序列化

名字本身就说明了两大特点：Fast（快速）—— 既指运行速度快，也指开发速度快。

为什么选择 FastAPI？

特性 说明
高性能 性能可与 Node.js 和 Go 媲美
自动数据验证 基于 Python 类型提示自动校验请求数据
自动生成文档 内置 Swagger UI 和 ReDoc，无需手写文档
类型提示 充分利用 Python 3.6+ 的类型系统
易于上手 代码简洁直观，学习曲线平缓

自动生成的交互式文档

这是 FastAPI 最方便的功能之一。启动服务后直接访问：

  • Swagger UI：http://127.0.0.1:8000/docs
  • ReDoc：http://127.0.0.1:8000/redoc

不需要写任何额外代码，FastAPI 根据路由和数据模型自动生成完整的 API 文档，Swagger UI 还支持在线调试——直接在网页里填参数、发请求、看结果。

核心概念速查

概念 代码示例 说明
GET 路由 @app.get("/path") 查询数据
POST 路由 @app.post("/path") 创建资源
路径参数 def fn(id: int) /users/42 → id=42
查询参数 def fn(name: str = "默认值") 自动解析 ?name=xxx
Pydantic 模型 class Item(BaseModel) 定义并校验请求体
嵌套模型 模型中引用另一个模型 支持复杂 JSON
cURL GET curl http://localhost:8000 命令行测试 GET
cURL POST curl -X POST -d '{...}' 命令行测试 POST

总结

FastAPI 最大的魅力在于：用最少的代码，做最多的事。

只需定义路由函数 + Pydantic 模型，参数解析、类型转换、数据校验、JSON 序列化、Swagger 文档全部自动搞定。Python 类型提示不再只是注释，而是真正参与到了运行时验证、文档生成和编辑器智能提示中。对于习惯 Flask 或 Django 的开发者来说，FastAPI 的体验会让你耳目一新。
